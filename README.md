## Project Overview

Taste of Uruguay is a content-driven landing page that introduces users to traditional Uruguayan cuisine, cultural values, and community-focused food experiences.

This project focuses on visual polish, usability, and inclusive design, simulating a real-world marketing or informational site.

## Features

- Hero section with clear call-to-action

- Featured dishes carousel with visual navigation

- Cultural storytelling section explaining Uruguayan food traditions

- Email sign-up form with selectable preferences

- Responsive layout across screen sizes

- Semantic HTML structure for accessibility

- Keyboard-navigable interactive elements

- Clear content hierarchy and readable typography
